# 虚拟资料自动赚钱机 (Virtual Docs Profit Theme)

一款专注于虚拟文档/文库资源变现的 WordPress 主题，集成文档管理、付费下载、会员体系、分销系统，开箱即用。

## 功能概览

### 核心功能

- **文档付费下载** — 支持单篇付费、VIP 免费下载，灵活定价
- **VIP 会员体系** — 多等级会员（月卡/季卡/年卡/永久），自动到期管理
- **微信扫码登录** — 公众号 OAuth 扫码登录（URL 二维码方案，无需用户关注）
- **合伙人分销** — 二级分佣（L1/L2），自动结算，提现管理
- **在线预览** — 123 网盘文档预览集成
- **AI 摘要** — DeepSeek AI 自动生成文档摘要

### 存储方案

- **腾讯云 COS** — 对象存储，私有读写 + 临时签名下载
- **123 网盘** — 网盘直链下载 + 在线预览

### 支付集成

- 支持对接主流支付（微信支付/支付宝），订单管理

### 后台管理

- **仪表盘** — 收入统计、订单概览、用户增长
- **会员管理** — 全用户列表、VIP/普通/过期分类筛选、搜索、手动续期
- **文档管理** — 批量上传、价格设置、下载统计
- **合伙人管理** — 佣金审核、提现处理、关系链查看
- **主题设置** — Codestar Framework 可视化配置面板

### 前端特性

- 响应式布局（Bootstrap 基础）
- 丰富的小工具系统（用户卡片、文档列表、VIP 推广、Hero 横幅等）
- 用户中心（个人资料、订单记录、下载记录、VIP 管理、合伙人面板）
- Font Awesome 图标库

## 安装部署

1. 将主题文件夹上传到 `wp-content/themes/virtual-docs-profit-theme/`
2. 后台「外观 → 主题」中启用
3. 进入「VDP 设置」完成基础配置：
   - 存储方式（COS / 123 网盘）
   - 支付接口
   - 会员套餐与定价
   - 社交登录（微信公众号 AppID/AppSecret）

## 微信登录配置

主题支持两种微信登录模式：

### 公众号模式（推荐）

适用于已认证的微信服务号：

1. 微信公众平台 → 设置与开发 → 功能设置 → 网页授权域名，填入你的网站域名
2. 主题设置 → 社交登录 → 填入服务号的 AppID 和 AppSecret
3. 模式选择「公众号」

PC 端扫码流程：生成 URL 二维码 → 用户微信扫码 → 微信内 OAuth 授权 → PC 端自动登录

### 开放平台模式

适用于已创建网站应用的微信开放平台账号：

1. 开放平台创建网站应用，填入授权回调域名
2. 主题设置填入开放平台 AppID 和 AppSecret
3. 模式选择「开放平台」

## 合伙人分销

- **二级分佣**：L1 直推佣金（默认 20%）+ L2 间推佣金（默认 5%）
- **冷冻期**：佣金产生后进入冷冻期（默认 7 天），防止退款刷单
- **提现方式**：微信 / 支付宝 / 余额抵扣
- **防作弊**：同 IP 注册标记、退款自动撤销佣金

## 数据库表

主题激活时自动创建以下自定义表：

- `wp_vdp_orders` — 订单记录
- `wp_vdp_memberships` — 会员记录
- `wp_vdp_partner_relations` — 分销关系链
- `wp_vdp_partner_commissions` — 佣金记录
- `wp_vdp_partner_withdrawals` — 提现记录
- `wp_vdp_wx_scene` — 微信扫码登录状态
